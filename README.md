### 项目技术架构
***
*  vue-cli
*  vue
*  vuex
*  vue-resource
*  vue-router
*  stylus
*  better-scroll
*  eslint
*  webpack

### 安装步骤
***
``` bash
# 安装依赖
npm install

# 启动服务 localhost:8080
npm run dev

# 打包
npm run build

# localhost:9000
node prod.server
```
